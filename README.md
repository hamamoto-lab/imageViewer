1. `data`ディレクトリに供覧したいデータを入れます

   ```

     |– sample01/
     |	|– file_01.png

     |	|– file_03.png
     |	|– file_04.png
     |	|– file_05.png
     |	|– file_06.png
     |– sample02/
     |	|– ︙

   ```
2. 以下のスクリプトを実行

```
streamlit run front.py
```
